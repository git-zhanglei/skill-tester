# Skill Certifier 优化头脑风暴

## 当前测试过程回顾

### 刚才发生了什么
1. 运行测试：`python3 scripts/certifier.py ../1688-shopkeeper`
2. 生成了 23 个测试案例
3. 执行了 Mock 模式测试
4. 生成了报告

### 暴露的问题
1. **测试案例太浅** - 只是字符串匹配，没有实际验证
2. **Mock 模式局限** - 模拟结果，不是真实执行
3. **报告信息不足** - 缺少具体的失败原因分析
4. **测试覆盖不全** - 缺少 API、数据、安全等维度

```python
def _spawn_session(self, task: str) -> Dict[str, Any]:
    """真实调用 OpenClaw"""
    try:
        # 调用 sessions_spawn
        result = sessions_spawn(
            task=task,
            timeout=self.timeout,
            runtime="subagent"
        )
        return self._parse_result(result)
    except Exception as e:
        return {'success': False, 'error': str(e)}
```
